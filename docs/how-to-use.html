インストール

Windowsの場合

Releases から「～-win.zip」をダウンロードします。 バージョンを選ぶ際は「バージョン番号の見方」を参考にしてください。
「ElectronShogi Setup ～.exe」を開きます。
インストールが成功するとElectron将棋が起動します。

Macの場合

Releases から「～-mac.zip」をダウンロードします。 バージョンを選ぶ際は「バージョン番号の見方」を参考にしてください。
「ElectronShogi-～.dmg」を開き、Electron将棋のアイコンをApplicationsへドラッグ&ドロップしてください。

バージョン番号の見方

Electron将棋のバージョン番号はセマンティックバージョニングに従っています。 不具合修正が入るとパッチバージョン（3番目の数）が上がります。その場合はできるだけ早くアップデートをしてください。 新機能が入るとマイナーバージョン（2番目の数）が上がります。新しい機能を使用したい方はアップデートをしてください。 大型アップデートの際はメジャーバージョン（1番目の数）が上がる場合があります。

エンジン登録

「エンジン設定」を開きます。
「追加」を押してエンジンの実行可能ファイルを選択します。
「設定」を押してエンジンの設定項目を表示します。
各項目を設定します。
※同じエンジンを複数追加した場合、それぞれに異なる値を設定できます。その場合は表示名に「検討用」「対局用」などを入れると判別しやすくなります。
最後に「保存して閉じる」を押します。（このボタンを押すまで各エンジンの設定は保存されません。）
