Electron将棋とは

Electron を使って作られたオープンソースの将棋アプリです。 Windows や MacOS など複数のプラットフォームで動作します。 誰でも開発に参加したり独自のバージョンを作成して配布することが可能です。
USIプロトコルに対応したエンジンを登録して対局や検討ができます。 同じエンジンを異なる設定でいくつでも登録でき、「水匠 対局用」「水匠 検討用」など用途に合わせた名前を付けることが可能です。

PCアプリ

  • 最新ベータ版
Windowsは末尾 -win.zip 、MacOSは末尾 -mac.zip のファイルをダウンロードしてください。
現在のバージョンでは、MacOS 版の複数起動をサポートしていません。
また、Ponder（相手番先読み）機能は実装されておらず、エンジン設定で有効化しても先読みは行われません。
今後のバージョンで予定している機能は 開発予定 を参照してください。

ブラウザアプリ

ブラウザ版アプリを開く
エンジンを使った対局やファイルの読み書きなど一部の機能は利用できません。
現在、スマホやタブレット等の縦長の画面はサポートしていません。（対応を検討中）
Chrome以外の動作は確認していませんが不具合報告は歓迎です。
