Electron将棋とは

Electronを使って作られたオープンソースの将棋アプリです。 Windows や MacOS など複数のプラットフォームで動作します。 誰でも開発に参加したり独自のバージョンを作成して配布することが可能です。
USIプロトコルに対応したエンジンを登録して対局や検討ができます。 同じエンジンを異なる設定でいくつでも登録でき、「水匠 対局用」「水匠 検討用」など用途に合わせた名前を付けることが可能です。

対応環境

  • Windows
  • MacOS
  • Linux
※Linuxのバイナリは用意していないためNodeJSの開発環境をセットアップしてください。
※その他の環境でもブラウザ版で一部の機能を利用可能です。

サポート

GitHubアカウントをお持ちの方はGitHub Issuesへお願いします。 設計に関する提案やPull Requestも歓迎です。
GitHubアカウントをお持ちでない方の連絡方法は準備中です。
※軽微ではないPRを作成する場合は必ず実装前にIssueでご相談ください。作業が競合したり設計方針を合意できないと、せっかく実装いただいた内容を取り込めない場合があります。
